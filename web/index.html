Noita 存档编辑器

玩家属性*

改移速翻倍时建议同调 run_velocity 与最大速度。

无敌

勾选/取消即时写入编辑缓冲。负 HP / 巨额 HP 两种取巧模式有副作用,暂不在 UI 提供。

受伤倍率*

1=正常,0=免疫,负值=受击回血。

法杖与法术*

法杖属性表单统一由右侧按钮一次应用全部法杖;点空槽添加法术,点已有法术编辑次数/Always Cast/删除,拖拽调整顺序(即时写缓冲;金色边框 = 永久附加)。

背包法术(inventory_full)

快捷栏法杖

特殊效果

添加/移除即时写入编辑缓冲;危险效果会红字警示。

    添加效果

    天赋

    注入 = 写入效果实体 + 运行旗标 + 拾取计数三件套;添加/移除即时写入编辑缓冲。

      添加天赋

      标记含义:无标记 = 与游戏内拾取等价;* = 缺少外观或数值微调,核心收益完整; ▲ 无代价 = 收益生效但没有原版的负面代价(比正常拾取更强); ⚠ 注入不完整 = 招牌能力由游戏脚本实现,注入只能拿到附带效果,不建议使用; 「不支持」= 整个天赋由脚本实现(如乐奇突变),存档层无法复刻。 选中任意天赋可在上方查看具体说明。

      药水与材料袋*

      编辑快捷栏容器所装材料与数量(约 1000 = 一满瓶),按容器逐个应用。材料框输入可搜索,须使用字典内的材料 id。

      物品栏道具*

      快捷栏道具行共 4 格(0–3)。被幽灵/腐蚀弄没的道具(如邪王真眼)选中后点添加即可补回空槽;满 4 件时添加禁用。进入编辑缓冲,需【写入存档】才落盘。

      世界状态*

      旗标与变换按整表替换;字段留空跳过。
